What Is Moltbot?

Moltbot (formerly Clawdbot) is an open-source, self-hosted AI assistant. You control where it runs, what it can access, and which channels it answers on.

Think of it as:

  • A private AI assistant
  • That lives on your own machine or VPS
  • And replies to you on WhatsApp, Telegram, Discord, Slack, and more

The Gateway is just the control center. The real product is the assistant itself.

How Does Moltbot Work? (Simple Explanation)

Messages from apps like WhatsApp or Discord go to the Gateway.

The Gateway:

  • Manages sessions
  • Routes messages to the correct agent
  • Controls tools, files, and actions

The agent then:

  • Thinks
  • Uses tools if needed
  • Sends the reply back to the same app

Everything stays under your control.


Installation (Recommended Way)

What You Need

  • Node.js version 22 or newer
  • macOS, Linux, or Windows (WSL2 strongly recommended)

Install Moltbot

npm install -g moltbot@latest
moltbot onboard --install-daemon


The onboarding wizard:

  • Sets up the Gateway
  • Creates your workspace
  • Connects channels
  • Installs essential skills

This is the official and recommended setup.


Quick Start Commands

# Start the Gateway
moltbot gateway --port 18789

# Send a message
moltbot message send --to +1234567890 --message "Hello from Moltbot"

# Talk to the assistant directly
moltbot agent --message "Plan my week"

Security: What You Must Know

Moltbot connects to real messaging apps. That means security matters.

Safe Defaults

  • Unknown users must pair before chatting
  • Messages from strangers are ignored by default
  • Group access requires explicit permission

Best Practices

  • Never expose the Gateway directly to the internet
  • Use allowlists for messaging apps
  • Run non-main sessions in Docker sandbox mode
  • Regularly run:
moltbot doctor


Moltbot is powerful ‚Äî and power must be handled carefully.
